Signatures

Die Einstellungen auf der Unterseite Digital Signatures beeinflussen sowohl den Befehl Quick Signature als auch Quick Verification. Wie im Wizard müssen Sie zunächst auswählen welche Ressource Sie signieren möchten und welchen Signaturtyp Sie verwenden wollen (nur enveloping und enveloped stehen zur Verfügung). Anschließend müssen Sie noch die zu verwendenden Algorithmen und einen Java Keystore mit dem Zertifikat auswählen sowie den Aliasnamen dieses Zertifikats angeben. Die Signature ID schließlich entspricht auch der ID nach der bei der Quick Verification gesucht wird. Auf fehlende Angaben werden Sie beim Aufruf von Quick Signature bzw. Quick Verification hingewiesen und können diese Angaben dann entweder nachholen oder die Operation abbrechen.
