Encryption Wizard

Mit dem Encryption Wizard können Sie auf komfortable und einfache Weise XML-Dokumente verschlüsseln.

Der Wizard besteht aus zwei Seiten die nacheinander ausgefüllt werden müssen. Nur wenn Sie alle Pflichtfelder einer Seite korrekt ausgefüllt haben können Sie auf die nächste Seite wechseln. Beachten Sie zu Ihrer Unterstützung die Textmeldungen oben im Assistenten. Der Wizard kann nur auf der letzten Seite zum Generieren der Verschlüsselung beendet werden.

Der Encryption Wizard wird über den Eintrag New Encryption... im Kontextmenü aufgerufen. Er besteht aus den Seiten Resource und Algorithms. Über das Hilfe-Icon links unten auf jeder Seite erhalten Sie eine kontextabhängige Kurzhilfe und können weitergehende Informationen abrufen.
