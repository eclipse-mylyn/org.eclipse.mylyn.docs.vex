Decryption Wizard

Mit dem Decryption Wizard können Sie auf komfortable und einfache Weise Ihre verschlüsselten XML-Dokumente wieder entschlüsseln und den Klartext wiederherstellen. Hierfür benötigen Sie die bei der Verschlüsselung gespeicherte Datei mit dem Schlüssel und den für diese Datei gewählten Verschlüsselungsalgorithmus.

Der Decryption Wizard wird über den Eintrag New Decryption... im Kontextmenü aufgerufen. Er besteht aus einer einzigen Seite Decryption. Über das Hilfe-Icon links unten auf jeder Seite erhalten Sie eine kontextabhängige Kurzhilfe und können weitergehende Informationen abrufen.
