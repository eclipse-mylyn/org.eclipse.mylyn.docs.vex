XML Security Tools User Guide

The XML Security Tools consist of the five parts Canonicalization, Signature Wizard, Signatures View, Encryption Wizard and Decryption Wizard. The focus is lying on the resource to secure, starting point therefore is the Eclipse WTP XML editor (or another supported XML editor) as well as the Navigator or Package Explorer view. All wizards therefore ask for the resource to secure first before more options for the signature or encryption have to be set.

Use one of the fast signing, verifying, en- and decrypting Quick Functions. The used settings have to be defined in the workspace preferences.
